Terminal

zsh / bash

Navigation

List all files and folders:

% ls
% ls [folder name]

-l show as list
-a show hidden
-1 show as single column

Change directory

% cd
% cd [folder name]
% cd .. back to parent folder
% cd ~ back to home folder

Show current path

% pwd

Clear console

% clear


File system

Create a file / folder:

% touch [file name]
% mkdir [folder name]

Move or rename a file / folder:

% mv [file or folder name] [target or new name]

Copy file:

% cp [file name] [target]

Delete file:

% rm [file name] [file2 name] [file3 name]

-i ask the user for confirmation before removing
-r recursive deletion for all files and sub-directories
-f force deletion

Delete empty folder:

% rmdir [folder name]

Show file content:

% cat [file name]


Applications

Nano editor:

% nano

Apache HTTP Server:

% sudo apachectl restart
% httpd -v check the current version
% httpd -t syntax test for configuration files


Processes

Show the current shell process:

% ps

-e show all running processes

Show process by process id:

% ps p [process id]

Terminate a process:

% kill [process id]









PID – process ID
TTY – terminal type
TIME – time spent on process running
CMD – command that launched the process


Curl

GET request

% curl https://myfakeapi.com/api/cars/3

More detailed:

% curl -v https://jsonplaceholder.typicode.com/albums/2

Save the output into a file

% curl -o emma.json https://reqres.in/api/users/3

Detailed request:

% curl -X [method] -H [header] -d [data]

POST request

% curl -X POST -d '{"name":"Neo","job":"Chosen One"}' https://reqres.in/api/users/
